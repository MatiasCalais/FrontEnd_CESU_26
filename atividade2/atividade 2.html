Lógica de Programação

O que é Lógica de Programação?

A lógica de programação é a maneira organizada de pensar para resolver problemas. Antes de escrever um programa em qualquer linguagem, é necessário entender como resolver o problema passo a passo.

A programação não começa digitando código, mas sim planejando a solução. Esse planejamento é feito através de algoritmos.

Algoritmo

Um algoritmo é uma sequência finita e organizada de passos que resolvem um problema. Ele deve ter início, meio e fim.

Exemplo de algoritmo simples para somar dois números: 

Início
   Ler número A
   Ler número B
   Soma = A + B
   Mostrar Soma
Fim

Variáveis

Variáveis são espaços na memória utilizados para armazenar valores. Elas podem guardar números, textos ou outros tipos de dados.

Exemplo: uma variável chamada "idade" pode armazenar o número 18.

Estruturas Condicionais

As estruturas condicionais permitem que o programa tome decisões. Elas utilizam condições para executar diferentes ações.

Exemplo: se a idade for maior ou igual a 18, mostrar "Maior de idade".

Estruturas de Repetição

As estruturas de repetição servem para executar um conjunto de comandos várias vezes enquanto uma condição for verdadeira.

Exemplos comuns são: for e while.
